# Hourglass

一个纯前端的 3D 物理沙漏模拟器（无第三方依赖）。

## 使用方式

1. 直接用浏览器打开 `index.html`，或在项目目录启动任意静态文件服务。
2. 在画布上按住拖拽可任意角度旋转沙漏。
3. 点击「翻转到对面」按钮会将沙漏转到回正角度的正对面。
4. 点击「重置沙漏」可完全重置角度与沙粒状态。
5. 拖动滑杆可实时调整沙粒数量（1000 - 26000）。
6. 画布支持双击翻转。

## 兼容性

- 桌面端：Chrome / Edge / Safari / Firefox
- 移动端：iOS Safari、Android Chrome（已处理触控、DPR、Safe Area 与横竖屏变化）
