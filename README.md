# 3D Hourglass (GitHub Pages Ready)

A lightweight Three.js experiment that renders a physically inspired 3D hourglass with thousands of animated sand grains. It is fully client-side and works on GitHub Pages—open `index.html` directly or host the repository with Pages.

## Features
- Glass body built from lathe geometry plus metallic frame and podium.
- ~5k sand particles with gravity, wind, pile formation, and gradual top-chamber depletion.
- OrbitControls for free camera rotation/zoom; double-click the scene to reset the view.
- Soft lighting, fog, and premium UI overlay for a gallery-like presentation.

## Development
No build step is required. Serve the root folder (e.g., `npx serve .`) or open `index.html` in a modern browser. Internet access is needed to load the Three.js CDN modules.
